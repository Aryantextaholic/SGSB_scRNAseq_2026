# SGSB midgut scRNA-seq analysis

This repository contains the R workflow used for SGSB (*Nezara viridula*) midgut scRNA-seq analysis (Seurat + Harmony + DoubletFinder + Monocle3).

## File
- `SGSB_final_midgut_scRNA_2026.R`

## Notes
- Large data objects (e.g., .rds, CellRanger matrices) are not included.
- Please edit file paths inside the script before running.
